The Guggenheim Alphabets

This publication is composed of a range of explorations for designing alphabet systems.An alphabet, by definition is “a set of letters or symbols in a fixed order, used to represent the basic sounds of a language; in particular, the set of letters from A to Z (Oxford).” Dealing not just with letter forms, an alphabet can essentially be defined as “basic elements in a system that combine to form complex entities” such as DNA genetic code, semaphore flag signals, and binary code.Based on a set of self-designed parameters, each student applied, tested, and developed an alphabet investigating how visual systems and language can define new contexts for meaning.


This alphabet was inspired by the Guggenheim Museum building designed by Frank Lloyd Wright in 1939. The spiral-shaped architectural frame informed the grid applied to the letterforms.
